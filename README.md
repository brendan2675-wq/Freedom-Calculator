# Freedom Calculator

The Freedom Calculator by Atelier Wealth — model your PPOR pay-down strategy and pay off your home in 10 years, not 30.

## Tech stack

- Vite
- React + TypeScript
- Tailwind CSS
- shadcn/ui

## Getting started

You'll need [Node.js](https://nodejs.org/en/) (LTS) and npm installed. Using `nvm` is recommended.

```sh
# 1. Clone the repo
git clone <YOUR_GIT_URL>

# 2. Move into the project directory
cd <YOUR_PROJECT_NAME>

# 3. Install dependencies
npm install

# 4. Start the dev server
npm run dev
```

## Available scripts

| Script           | Purpose                          |
|------------------|----------------------------------|
| `npm run dev`    | Start the local dev server       |
| `npm run build`  | Production build                 |
| `npm run preview`| Preview the production build     |
| `npm run lint`   | Run ESLint                       |
| `npm test`       | Run the Vitest test suite        |

## Deployment

Build the project with `npm run build` and deploy the contents of the `dist/` folder to any static host (Vercel, Netlify, Cloudflare Pages, S3 + CloudFront, etc.).

## Custom domain

Configure your custom domain at the DNS level on your chosen host.
